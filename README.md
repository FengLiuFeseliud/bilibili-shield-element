# bilibili-shield-element

基于 [Plasmo](https://www.plasmo.com/) 的 bilibili 元素屏蔽浏览器插件

## 安装

因为本插件还没有上架商店, 请解压安装

在 [releases](https://github.com/FengLiuFeseliud/bilibili-shield-element/releases) 下载插件, 之后在插件页打开开发者模式, 将插件解压, 点击 `加载已解压的扩展程序` 加载插件

## 运行开发环境

```bash
npm run
# OR (推荐)
pnpm run
```
